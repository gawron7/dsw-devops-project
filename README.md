Aplikacja pogodowa w React pokazująca pogodę dla polskich miast.

## Funkcje
- Pogoda aktualna i prognoza 5-dniowa
- Wyszukiwanie miast
- Ulubione miasta (localStorage)
- Jednostki temperatury (°C/°F/K)
- Responsywny design
- Integracja z OpenWeatherMap API (opcjonalna)

## Uruchomienie
```bash
npm install
npm run dev
```
Otwórz: http://localhost:3000

## API (opcjonalne)
1. Klucz z [OpenWeatherMap](https://openweathermap.org/api)
2. Utwórz `.env`: `VITE_OPENWEATHER_API_KEY=twój_klucz`
3. Restart: `npm run dev`

Bez API działa z danymi demo.
